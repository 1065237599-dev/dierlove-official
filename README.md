# Dierlove Official Brand B2B Website

This is the official B2B website for Dierlove, a premier lingerie and swimwear manufacturer based in Quanzhou.

## Features
- **High-end Luxury Visuals**: Optimized for premium Western markets.
- **B2B Focused**: Clear MOQ (50/200 PCS) and custom solutions.
- **Mobile Responsive**: Fully optimized for mobile buyers.
- **Conversion Driven**: Integrated WhatsApp and 24h Quote inquiry system.

## Deployment
Hosted on Cloudflare Pages via GitHub.

## Technical Stack
- HTML5 / Tailwind CSS
- Framer Motion
- Google Fonts (Playfair Display, Montserrat)
